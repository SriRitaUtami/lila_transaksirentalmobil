PROJECT TITLE : RENTAL MOBIL
PURPOSE OF PROJECT : FINAL LABORATORIUM PEMROGRAMAN BERORIENTASI OBJEK
VERSION OF DATE : 30 DESEMBER 2020

DESCRIPTION :
Deskripsi program :
         Tujuan pembuatan program rental mobil dengan java ini adalah supaya kita dapat mengetahui hasil dari perentalan kendaraan mobil. Dan program ini lebih mudah di bandingkan dengan cara manual yang membutuhkan waktu yang lama untuk membuat laporannya.
         Dengan program ini terdapat banyak tipe data yang di gunakan seperti, string , integer dan sebagainya. Dan menggunakan nama variabel diantaranya  kode mobil, jenis mobil, lama rental, dan total biaya atau harga yang diperlukan untuk mengetahui harga yang sudah di tetapkan oleh toko rental tersebut.
         Dimana dalam perentalan mobil ini memerlukan suatu program yang dapat mengolah data dengan baik dan efisien. Dengan program ini kita dapat mengolah data-data yang talah disediakan seperti  macam-macam jenis mobil, Sehingga dapat memproses dalam pengolahan data yang ada dalam perentalan mobil dengan menggunakan  pemrograman java  sehingga  memudahkan kita dalam perentalan mobil. 
Pada program ini terdapat data data yang diminta kepada seseorang Ketika ia hendak merental yang berupa identitas diantaranya nama,no hp,alamat,no ktp,dan lain lain. Kemudian Ketika mobil yang ingin di rental akan ditentukan jenis, tanggal, lama rental sehingga aka nada total biaya yang perlu dibayar. Akan ada denda setiap keterlambatan Ketika pengembalian mobil tergantung berapa hari keterlambatan. Kemudian juga akan ada biaya ganti rugi Ketika kondisi mobil Ketika dikembalikan tidak seperti Ketika di rental pada awal.
